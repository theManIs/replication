   Структура данных - программная еденица, позволяющая хранить и обрабатывать множество однотипных и/или логически связанных данных в вычислительной технике. Для добавления поиска, изменения и удаления данных структура данных представляет собой некоторый набор функциий, составляющих её интерфейс.
   Список структур данных: список, массив, битовое поле, параллельный массив, ассоциативный массив, хэш-таблица,
стэк, очередь, дек, буферное окно, граф, таблица, дерево, двоичное дерево, куча.
   Граф - совокупность непустого множества вершин и наборов пар вершин (связей между вершинами).
   Двоичное дерево поиска - это древовидная структура данных для которой характерны такие признаки: оба поддерева являются двоичными деревьями поиска, у всех узлов левого поддерева произвольного узла Х значения ключей данных меньше, нежели значение ключа данных самого узла Х, значения ключей данных у всех узлов правого поддерева больше, нежели значения ключа данных узла Х.
   Куча - название структуры данных, с помощью которых реализована динамически распределяемая память приложения.
   Стек - абстрактный тип данных, представляющий собой список элементов, организованных по принципу "последним пришел - первым ушел" (лифо).
   Очередь - абстрактный тип данных с дисциплиной доступа к элементам "первый пришел - первый ушел" (фифо).
   Дэк - структура данных, в которой элементы можно добавлять и удалять как в начале, так и в конце очереди.
   Параллельный массив - структура данных для представления массива записей, которая физически состоит из отдельных однотипных массивов одинаковой длины для каждого из полей записи.
   Буферное окно - динамический массив, который позволяет эффективно проводить вставку и удаление элемента в некоторой области.
   Динамический массив - массив, размер которого при необходимости может меняться.
   Алгоритм - набор инструкций, описывающих порядок действий исполнителя для достижения результата решения задачи за конечное число действий, при любом наборе исходных данных.
   Виды алгоритмов: механические, гибкие, вероятностные (стохастические), эвристические, линейные, разветвляющиеся, циклические, вспомогательные (процедура).
   Тезис Чёрча - Тьюринга - любой конечный физический процесс, не использующий аппарат, связанный с неприрывностью и бесконечностью, может быть вычислен физическим устройством.
   Класс сложности - множество вычислительных задач, одинаковых по сложности и вычислению. Для его определения используется формула O(f(n)), n - объём входящих данных.
